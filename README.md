# Image Blur Tool

A simple tool to select and blur areas in an image using Tkinter and OpenCV.

## Usage

1. Load an image.
2. Select areas to blur (either circles or rectangles).
3. Click "Blur Areas" to blur the selected areas.
4. Save the blurred image.

## Requirements

- Python 3.x
- Pillow
- OpenCV
